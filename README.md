# Claude Code Skills

A collection of custom skills (slash commands) for [Claude Code](https://docs.anthropic.com/en/docs/claude-code) that automate common development workflows — committing & pushing, creating PRs, bumping versions across project files, and stress-testing plans through interrogation.

| Skill | Command | What it does |
|-------|---------|--------------|
| **commit-push** | `/commit-push` | Commit + push in one step |
| **commit-pr** | `/commit-pr` | Commit on a new branch + push + open PR |
| **bump-patch** | `/bump-patch` | `1.1.2` → `1.1.3` across all version files |
| **bump-minor** | `/bump-minor` | `1.1.2` → `1.2.0` across all version files |
| **bump-major** | `/bump-major` | `1.1.2` → `2.0.0` across all version files |
| **interrogate** | `/interrogate` | Relentless Q&A to stress-test a design or plan |
